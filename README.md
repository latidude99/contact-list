# General Info
My first foray into web apps world. Written after finishing a Spring Framework course mainly to practise things learnt
on the course. It is as the name suggests a list or database of contacts with user registration and criteria based 
search functionality.

# Technologies
- Java JDK 1.8
- Maven 3.60
- Apache Tomcat 8.0
- Sprin Boot 1.5 with Spring Security 4, Spring Validation and Spring Data
- Hibernate ver.
- Thymeleaf 2 with Thymeleaf Extras
- Eclipse / STS 3.9 , unit tests in IntelliJ IDEA. 
- Google Maps iFrame
- JUnit5 5.4.1
- AssertJ 3.12.2

**Plugins and libraries**
- Maven Compiler plugin 3.8.0
- Maven Site plugin version 3.6
- Maven Surefire plugin  version 2.22.0
